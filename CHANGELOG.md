# Changelog

## [1.1.0] — 2026-03-11

### Добавлено
- 🔊 `--sound` — звук когда Claude Code заканчивает отвечать (macOS + Linux)
- 🔔 `--notify` — системное уведомление по завершению
- 🎲 `random` — случайный пак фраз при каждом запуске
- ✏️ `--add "фраза"` — добавить свою фразу к текущим
- 🚀 `--all-features` — установить всё одной командой
- 🔄 `--reset` — сбросить к стандартным настройкам Claude Code
